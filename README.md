# Willow & Thorn Florist Demo 🌸

A modern florist business website built with Django and deployed on a live Linux VPS environment.

This project was created as a realistic small-business demo to showcase full-stack web development, responsive UI design, backend functionality, and real-world deployment workflows.

## Features

* Responsive modern UI
* Bouquet categories & filtering
* AJAX-powered sorting and category updates
* Featured product carousel
* Django admin panel
* Dynamic database-driven content
* Image uploads with WebP optimization
* HTTPS secured custom domain
* Production deployment with Nginx + Gunicorn
* Linux VPS hosting on DigitalOcean

## Tech Stack

* Python
* Django
* SQLite
* HTML / CSS / JavaScript
* Nginx
* Gunicorn
* DigitalOcean VPS
* Git & GitHub

## Purpose

This project is part of a growing portfolio focused on modern websites for local businesses such as florists, salons, garages, restaurants, and service-based companies.

The goal was to simulate a real client project from development to production deployment, including:

* server setup
* domain configuration
* SSL certificates
* static file handling
* Git-based deployment workflow
